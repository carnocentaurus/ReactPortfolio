# Pokédex
Interactive Pokémon information viewer using the PokéAPI.

## Features
- Random Pokémon generator
- Search by name or ID (1-1025)
- Display stats: height, weight, and type
- Official Pokémon sprites

## API
Uses [PokéAPI](https://pokeapi.co/) - a free RESTful Pokémon API.

## Technologies
- React + TypeScript
- Fetch API for data retrieval
- Tailwind CSS for styling
